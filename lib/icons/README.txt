To create the desktop shortcut to launch Memoranda with MS-Windows, 
do the following:

	1) Right-click the desktop and select "New -> Shortcut".
	2) In "Create Shortcut" dialog box press "Browse" and select the 
	   "memoranda.bat" file in the installation directory.
	3) Press "Next", enter "Memoranda" and press "Finish".
	4) Rignt-click the new shortcut and select "Properties"
	5) In "Shortcut" tab press "Change Icon". Press "Ok" in confirmation box.
	6) In "Change Icon" dialog box press "Browse", select 
	   "lib/icons/memoranda.ico" file and press "Ok".
	7) Press "Ok" to close the "Properties" dialog box.

To add the shortcut into the Windows "Start menu":
	
	1) Start the Windows Explorer and open the 
	   "C:\Documents and Settings\All Users\Start Menu\Programs" folder (to
	   place the shortcut into public start menu) or 
	   "C:\Documents and Settings\{YOURNAME}\Start Menu\Programs" folder (to
	   place the shortcut into your personal start menu).
	2) Select "File -> New -> Folder" to create a new folder and name it 
	   "Memoranda". 
	3) Copy and paste the Memoranda shortcut from desktop into this folder.
